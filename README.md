# GraphQL Server

Example to learn GraphQL. Based on the [Midudev's](https://www.youtube.com/@midudev) [complete course](https://www.youtube.com/playlist?list=PL2i4CEznA8jWqp8nsonAjddiN4XctqIuh) (in Spanish).

## Run the project

```bash
npm install

cp _.env .env

npm run json-server

npm start
```
